# Fitland RPG — PWA

## Deploy en Vercel (3 pasos)

1. Sube esta carpeta a un repo de GitHub
2. Ve a vercel.com → New Project → importa el repo
3. Vercel detecta React automáticamente → Deploy

## Instalar en Android
1. Abre la URL de Vercel en Chrome
2. Menú (3 puntos) → "Añadir a pantalla de inicio"
3. ¡Listo! Aparece como app instalada con ícono

## Instalar en iPhone
1. Abre la URL en Safari
2. Botón compartir → "Añadir a pantalla de inicio"

## Estructura
- src/App.jsx — toda la lógica del juego
- public/manifest.json — config PWA
- public/sw.js — service worker (offline support)
